Getting Started

Stratum Protocol: 
mining.alqo.org:4033
The ALQO Stratum is choosing one of our two main-nodes that fits your connection the best. However if you want to choose by yourself make sure to use the following stratum-domains:
Stratum Protocol EU: 
eu.mining.alqo.org:4033
Stratum Protocol US: 
us.mining.alqo.org:4033
Usage: 
-a quark -o stratum+tcp://mining.alqo.org:4033 -u {WALLET-ADDRESS}
NVIDIA - CCMiner [preconfigured] https://pool.alqo.org/ccminer.zip
AMD - SGMiner [preconfigured] https://pool.alqo.org/sgminer.zip
The ALQO Pool is adjusting the difficulty of your miner automatically.
If you somehow want to set the difficulty by your own you use set it using the password parameter. Example (0.2 Difficulty): 
-a quark -o stratum+tcp://mining.alqo.org:4033 -u {WALLET-ADDRESS} -p d=0.2
